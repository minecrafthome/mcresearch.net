The Panorama Project

to find the seed for the Panorama found from 1.8 to 1.12.2

 

What is this Project?

     This project branched off the project to find the pack.png seed. The Paranoma project goal is to find the seed of the paranoma in the main menu. Please join the Minecraft Research discord to contribute to this project and others.

What we know

Overview

  • The panorama existed from 1.8 prerelease 1 to 1.12.2.
  • The images were taken with 90 FOV.
  • The panorama was taken during the development of 1.8 therefore some features exist and some do not.
  • More information in Details

How Can I Help?
